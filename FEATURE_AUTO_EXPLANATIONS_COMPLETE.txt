🎯 HOW IT WORKS

1. SYSTEM ANALYZES CHANGE TYPE
   → Items increased? Show item-increase explanation
   → Items decreased? Show item-decrease explanation  
   → Only amount changed? Show amount-change explanation
   → Both changed? Show combined explanation
   → Negative item (deduction)? Special deduction explanation

2. CATEGORY-SPECIFIC MESSAGES
   → ילדי ח"מ (Children): "נוספו ילדים חדשים או תשלומי רטרו نوספים"
   → עוזרות (Helpers): "נוספו משרות חדשות או תשלומי רטרו"
   → גננות (Kindergarten deductions): "נוספת גננת עובדת מדינה שובצה לרשות"
   → Generic categories: "ייתכן הוספת משרות חדשות או פרמטרים"

3. CPA OVERRIDE (Custom Explanations)
   → If CPA has manually entered explanation → Show that (in italics)
   → Otherwise → Show auto-generated explanation

🔄 EXPLANATION TYPES (All in Plain Hebrew)

SCENARIO 1: ITEMS INCREASED
────────────────────────────
If: 24 → 28 items (+4 items)
Show: "4 פריטים נוספו — ייתכן רישום ילדים חדשים או תשלומי רטרו נוספים"
      (4 items added — likely new children registration or additional retroactive payments)

SCENARIO 2: ITEMS DECREASED  
────────────────────────────
If: 28 → 24 items (-4 items)
Show: "4 פריטים הוסרו — ייתכן סגירת משרות או סיום תשלומים"
      (4 items removed — likely closed positions or ended payments)

SCENARIO 3: AMOUNT INCREASED ONLY
──────────────────────────────────
If: Same items, amount ↑ (₪100,000 increase = 10%)
Show: "הסכום עלה ב-₪100,000 (10%) — ייתכן עדכון עלות לילד או שינוי אחוז השתתפות"
      (Amount increased by ₪100,000 — likely cost-per-child update or participation % change)

SCENARIO 4: AMOUNT DECREASED ONLY
──────────────────────────────────
If: Same items, amount ↓ (₪100,000 decrease = 10%)
Show: "הסכום ירד ב-₪100,000 (10%) — ייתכן עדכון עלות לילד או שינוי אחוז השתתפות"
      (Amount decreased — same reasoning)

SCENARIO 5: BOTH ITEMS AND AMOUNT CHANGED
───────────────────────────────────────────
If: 24 → 28 items AND amount ↑ 17.7%
Show: "4 פריטים נוספו והסכום עלה ב-17.7% — ייתכן שינוי כמותי וגם בתעריפים"
      (4 items added and amount increased — both quantity and rate changes)

SCENARIO 6: DEDUCTIONS (NEGATIVE ITEMS)
───────────────────────────────────────
If: topic_code = '33' (גננות - Kindergarten deductions) and items change
Show: "הניכוי גדל ב-X פריטים — ייתכן גננת עובדת מדינה נוספת שובצה לרשות"
      (Deduction increased — likely additional state-employed kindergarten teacher assigned)
